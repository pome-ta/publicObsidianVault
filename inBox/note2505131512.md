## CDN で実行

`<body>` 内で、`<script>` 呼ぶと出る

```html
<body>
  <script src="./js/sketchMain.js"></script>
</body>
```

`type="module"` するとダメ（そりゃそうか）

```html
<script type="module" src="./js/sketchMain.js"></script>
```
